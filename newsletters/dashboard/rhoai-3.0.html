Hardware Profiles (General Availability)

GPU configuration without the complexity

Hardware Profiles are now GA! Configure GPU, CPU, and memory allocations with pre-built templates instead of writing YAML. Think of them as presets for different hardware needs.

Available Profiles:

Profile CPU Memory GPU Best For
Small 2 cores 8 GB - Development, testing
Medium 4 cores 16 GB 1x NVIDIA T4 Training small models
Large 8 cores 32 GB 1x NVIDIA A100 Training large models
XLarge 16 cores 64 GB 4x NVIDIA A100 Distributed training

Create custom profiles in Settings ‚Üí Hardware Profiles, or use profiles directly when creating workbenches or deploying models!

Model Deployment Wizard

Deploying models is easier than ever

The new deployment wizard walks you through every step of deploying a model, from selection to going live.

The 3 Simple Steps:

Step 1: Select Model Details

Choose model runtime from a list and add a connection to get the stored model.

Step 2: Configure Model Deployment

Set deployment name, select hardware profile, define autoscaling parameters, and set environment variables.

Step 3: Advanced Settings & Deploy

Add authentication, configure playground settings, review and confirm deployment.

After Deployment:

Get your endpoint URL, test with built-in tools, monitor request counts and latency, and adjust replicas as needed!
